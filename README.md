# Conversor de Moedas em Java

Este é um projeto simples de **conversor de moedas** desenvolvido em **Java**, utilizando a **ExchangeRate API** para obter as taxas de câmbio em tempo real e a biblioteca **Gson** para manipular o formato JSON da resposta.

---

## Funcionalidades

- Converte valores entre **6 moedas pré-definidas**:
  - ARS - Peso Argentino  
  - BOB - Boliviano da Bolívia  
  - BRL - Real Brasileiro  
  - CLP - Peso Chileno  
  - COP - Peso Colombiano  
  - USD - Dólar Americano  

- Consulta a API ExchangeRate em tempo real.  
- Exibe o valor convertido com base na taxa atual.  
- Código modular, dividido em métodos para facilitar a manutenção.

---

## Tecnologias utilizadas

- **Java 17+**
- **Gson** (para leitura e manipulação de JSON)
- **HttpClient** (para fazer requisições HTTP)
- **ExchangeRate API** ([https://www.exchangerate-api.com/](https://www.exchangerate-api.com/))
